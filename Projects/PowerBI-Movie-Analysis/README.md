# Power BI Movie Analysis

## Project Overview

This project focuses on analyzing movie datasets using Power BI dashboards and visualizations.

The dashboard helps in understanding:
- Movie ratings
- Genre trends
- Popular movies
- Performance insights
- Audience analysis

## Tools Used

- Power BI
- Excel
- Data Visualization Techniques

- Interactive dashboard
- Genre-wise analysis
- Ratings comparison
- Trend visualization
- Business insight generation
